# Basic Pulseq Tutorial
## The tutorial is written by:
* Python within Jupyter Notebooks on Google Colab.
* Matlab translated from the Python source code by ChatGPT.    
This talk will focus on the tutorial based on Jupyter Notebook.

## The tutorial contains three sequences:
* s01_fid: FID.
* s02_fid2gre1d: from FID to 1D GRE.
* s03_gre1d2gre2d:from 1D GRE to 2D GRE.

## Necessary technical preparation for the tutorial on Jupyter Notebook
* Provided hands-on exercises and pulse sequence code snippets are in Python programming language.
* The hands-on exercises are based on running Python code in Notebooks within the Google Colab environment, therefore, no additional software is required.    
* A **Google account** and a **laptop or tablet connected to the Internet** are required to run Notebooks in Google Colab.
